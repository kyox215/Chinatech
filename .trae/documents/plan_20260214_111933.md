# 更新回收模块 UI 设计

我将对 `components/recycling-app.tsx` 中的“回收报价”卡片进行重新设计，使其更紧凑，并采用用户要求的“简约白色”风格。

## 修改计划

1.  **卡片容器 (Card Container)**:
    *   移除深色背景 `bg-slate-900`，改为默认白色背景 (`bg-card` 或 `bg-white`)。
    *   移除深色文本 `text-white`，使用默认文本颜色。
    *   简化边框和阴影，移除 `border-slate-800`，使用更轻的阴影。

2.  **头部区域 (Header)**:
    *   **紧凑化**: 将 `pb-6` 减少为 `pb-4`。
    *   **样式调整**:
        *   标题颜色从 `text-slate-400` 改为 `text-muted-foreground`。
        *   价格字体大小从 `text-7xl` 缩小为 `text-5xl`，颜色调整为 `text-green-600` 以适应浅色背景。
    *   **预测标签**:
        *   移除深色模式特有的半透明背景 `bg-white/5`。
        *   使用 `bg-muted` 或 `variant="secondary"` 的 Badge 样式。
        *   调整文字颜色以保证对比度。

3.  **内容列表 (Content)**:
    *   移除 `bg-slate-900`。
    *   **列表项样式**:
        *   标签文本改为 `text-muted-foreground`。
        *   数值文本改为 `text-foreground` 或 `font-bold`。
        *   分隔线颜色从 `border-white/5` 改为标准的 `border-border` 或 `border-dashed`。
    *   **扣款项颜色**: 调整红色、黄色等警告色的深浅，使其在白色背景下清晰可见（例如 `text-red-300` -> `text-red-600`）。

4.  **底部区域 (Footer)**:
    *   移除深色背景，使用 `bg-muted/5` 或纯白。
    *   调整按钮高度，使其更精致 (`h-14` -> `h-11` 或 `h-12`)。

## 预期效果
右侧的报价卡片将变为白色简约风格，视觉上更轻盈，尺寸更小巧，不再显得笨重或“碍事”。
