# 连接 Supabase 数据库 (Connect Supabase)

我将使用您提供的凭据配置项目以连接到 Supabase 数据库。

## 实施步骤

1.  **配置环境变量 (Configure Environment Variables)**
    *   创建 `.env.local` 文件，安全存储您的 API 密钥和数据库连接字符串。
    *   将提供的密钥映射为标准的 Next.js 环境变量名：
        *   `NEXT_PUBLIC_SUPABASE_URL`: 用于前端连接。
        *   `NEXT_PUBLIC_SUPABASE_ANON_KEY`: 用于前端连接的公共密钥。
        *   `POSTGRES_PRISMA_URL`: 用于后续可能使用的 Prisma 连接池。
        *   `SUPABASE_SERVICE_ROLE_KEY`: 用于服务端管理操作。

2.  **安装依赖 (Install Dependencies)**
    *   安装官方 Supabase 客户端库：`@supabase/supabase-js`。

3.  **初始化 Supabase 客户端 (Initialize Client)**
    *   创建 `lib/supabase.ts` 文件，初始化并导出 `supabase` 客户端实例，供全站调用。

4.  **连接确认**
    *   完成配置后，项目即具备了读写数据库的能力。
    *   (注：您当前的“回收模块”使用的是硬编码数据，连接建立后，我们后续可以将这些数据迁移到数据库中。)
