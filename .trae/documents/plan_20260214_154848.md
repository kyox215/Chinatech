# UI 优化方案：头部自动收纳与侧边栏清理

我将严格按照您的要求更新应用：
1.  **侧边栏**: 移除 "文件"、"项目"、"学习"、"社区" 和 "素材"。
2.  **头部栏**: 主头部 ("Designali") 和 应用头部 ("智能回收报价系统") 将在下滑时自动收起，且**仅在回到顶部时显示**。

## 实施步骤

### 1. 侧边栏清理 (`components/creative.tsx`)
- 修改 `sidebarItems`，仅保留 "首页" 和 "应用"。

### 2. 主头部自动收纳 (`components/creative.tsx`)
- 添加状态 `showMainHeader` (默认 `true`)。
- 使用 `framer-motion` 的 `motion.header` 包裹 `<header>` 元素。
- 根据 `showMainHeader` 状态通过动画控制 `height` 和 `opacity`。
- 将 `setShowMainHeader` 作为属性传递给 `RecyclingApp` 组件。

### 3. 回收应用与本地头部 (`components/recycling-app.tsx`)
- 更新组件定义，接收 `setMainHeaderVisible` 属性。
- 创建本地状态 `showAppHeader`。
- 在可滚动容器 (`lg:col-span-8 ...`) 上添加滚动监听器。
- **逻辑**:
    - 当 `scrollTop > 20` (向下滑动离开顶部): 隐藏两个头部 (主头部 & 应用头部)。
    - 当 `scrollTop <= 20` (回到顶部): 显示两个头部。
- 使用 `motion.header` 包裹本地 `<header>` ("智能回收报价系统...") 以实现同步动画效果。

该方案确保用户在浏览内容时界面更加清爽（全屏体验），只有当用户需要操作导航或查看标题并回到顶部时，头部才会重新出现。
