# 新增 128GB 存储选项

我将更新 `components/recycling-app.tsx` 中的存储配置，将原本合并的选项拆分，单独增加 128GB 选项。

## 实施步骤

1.  **更新 `storageTiers` 数据**:
    *   将原有的 `{ label: "64G / 128G", value: 0 }` 拆分为两个独立选项：
        *   `{ label: "64G", value: 0 }`
        *   `{ label: "128G", value: 8 }` (设定一个介于 0 和 256G(15) 之间的中间值)。
    *   保留现有选项：
        *   `256G` (Value: 15)
        *   `512G` (Value: 30)
        *   `1TB` (Value: 50)

2.  **优化 UI 布局**:
    *   调整存储按钮的网格布局以更好地容纳 5 个选项。
    *   将 `md:grid-cols-4` 修改为 `md:grid-cols-5`，使所有选项在桌面端能整齐地排在一行。

## 预期效果
用户现在可以明确选择 "128GB" 容量，且界面布局会自动适应显示 5 个存储选项。
